# Mini-Uber 

A simple ride-hailing API built with FastAPI and PostgreSQL.

## Features
- Create ride requests
- Store ride data in PostgreSQL
- RESTful API endpoints
- Client SDK for easy integration

## Setup

### Prerequisites
- Python 3.8+
- PostgreSQL
- PgAdmin4 (optional)
